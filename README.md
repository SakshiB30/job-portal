About the Project ✨

Job Portal is a beginner-friendly full-stack web app built with React (frontend) and Spring Boot (backend) using MongoDB.

💼 For Job Seekers: Browse and explore job opportunities
🏢 For Employers: Post jobs and manage applications

⚡ Work in progress — more cool features coming soon! 🚀

Features (Coming Soon) 🌟
📝 User registration and login
👤 Profile creation and editing
🔍 Job search and filters
📄 Apply for jobs
🏢 Employer dashboard for posting jobs
